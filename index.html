Welcome to Brew Tracker

Connect your Firebase account to sync data across all devices.

1. Go to console.firebase.google.com → Create project → disable Analytics.

2. Build → Firestore Database → Create → europe-west → Start in test mode.

3. Project Settings ⚙️ → Your apps → </> → Register app → copy the firebaseConfig and paste below.

Your data stays in your Firebase account. Anthropic has no access.
